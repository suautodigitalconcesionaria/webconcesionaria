# AUTOBA Web

Este repo ahora incluye **versión estática HTML** lista para GitHub Pages.

## Archivos principales para GitHub Pages

- `index.html` (home)
- `stock.html`
- `detalle.html`
- `financiacion.html`
- `contacto.html`
- `styles.css`
- `app.js`

Con esto podés publicar directo en GitHub Pages sin build.

## Publicar en GitHub Pages

1. Subí estos archivos al branch `main`.
2. En GitHub: **Settings → Pages**.
3. En **Source**, elegí **Deploy from a branch**.
4. Seleccioná `main` y `/ (root)`.
5. Guardá y esperá el link público.

## Nota

El código Next.js queda en el repo como alternativa, pero para abrir páginas desde GitHub Pages se usan los `.html` de raíz.
